# Netflix_AttentionSpans

Every 6 months, Netflix publishes their Engagement Report and reveal which movies had the highest viewership on their platform for half of the year. Dedicated movie fans, data scientists, and probably even Netflix themselves hope to gain some sort of insight on what movies are trending: what makes a movie popular?
Now, the data published in this report is sparse. To the report, the movie is effectively three things: a title, a runtime, and a number of views. But if we enhance this dataset with genre tags from TMDB, what can we learn? Attention Spans examines genre-specific movie trends by comparing runtime to Netflix viewership in the first half of 2025.


Made in partnership with Netflix for Parsons MSDV Data Visualizations and Info Aesthetics, Fall 2025.
